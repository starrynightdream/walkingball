## 未归类
- 人的本质：一切社会关系的总和
- 必然存在于偶然之中
- 必然使用偶然为之开路
- 内因 : 内部矛盾
    + 事物存在的基础
    + 一事物区别于其它事物的内在本质
    + 事物变化的根据，它规定着事物发展的方向
- 外因 : 事物之间的相互联系、相互影响
    + 事物变化的条件
    + 能够加速活延缓甚至暂时改变事物发展的进程
    + 必须通过内因而起作用
- 度 ： 事物保持自己质的稳定性的数量界限
    + 即事物的范围、幅度和限度
- 否定
    + 辩证的否定：扬弃
        * 新事物对旧事物既批判又继承，既克服其消极因素又保留其积极因素。
        * 是新旧事物联系的环节
        * 否定是事物的自我否定，是事物内部矛盾运动的结果。
    + 形而上学的否定
        * 否定是外在的否定，主观任意的否定
        * 否定是绝对的否定，是不包含肯定的否定。
- 真理
    + 具有客观性，是绝对性和相对性的辩证统一
- 实践
    + 与动物本能的、被动的适应性活动不同，人的实践活动是一种有意识、有目的的活动。
        * 目的性是能动性的主要表现。
- 实践是
    + 认知的起点，也是认知的归宿
    + 认识发展的动力
- 认识是主体和客体相互作用的产物。
    + 非理性因素对理性因素既起着
- 理性认识是认识的高级阶段，具有抽象性和间接性的特点。
    + 它以反映事物的本质为内容，因而是深刻的。
- 直接性是感性认识的突出特点。
- 理性认知以感性认知为基础。
- 感性向理性的飞跃
    + 投身实践、深入调查
    + 理性思考、去粗取精
- 实践最终一定能鉴别认识的真理性

## 矛盾
- 特殊性
    + 各个具体事物的矛盾及每一个矛盾的各个方面在发展的不同阶段上各有其特点 
- 矛盾的同一性的有条件的、相对的，矛盾的斗争性是无条件的，绝对的。
